# SingleTune

**SingleTune** is a music playback and management application written in Java.  
It allows users to play and organize audio tracks locally, providing basic controls and library features.

---

## Overview

SingleTune is designed as a simple yet functional Java-based desktop application for managing and listening to music files. Itâ€™s ideal as a learning project or foundation for more advanced features like playlists, metadata editing, and enhanced UI.

---

## Features

- Play local audio files
- Playback controls (play, pause, stop)
- Track navigation (next, previous)
- Local music library browsing
- Basic management of songs


## Getting Started

To run SingleTune locally:

### 1. Clone the repository
```bash git clone https://github.com/pcsolano/SingleTune.git```
### 2. Open the project in your preferred IDE as a Java project.

### 3. Build and run the application using the IDE tools or from the command line:
javac -d out src/**/*.java
java -cp out Main
